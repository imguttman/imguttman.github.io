HTML and CSS Cheatsheet

Key Definitions

  • HTML:

    Acronym for Hypertext Markup Language, a standardized system for tagging text files to achieve font, color, graphic, and hyperlink effects on World Wide Web pages..

  • CSS:

    Acronym for Cascading Stylsheets, a style sheet language used for describing the look and formatting of a document written in a markup language.

  • Elements:

    Elements are the building blocks of HTML documents. An element is the content written in between a start tag, with an end tag.

  • Tags:

    A command inserted in a document that specifies how the document, or a portion of the document, should be formatted. Tags are written between < > and can vary in function: some are necessary for beginning an HTML document and others can be used to start a new paragraph.

  • id:

    An id selector is written in CSS (as #id name) and then assigned to a single HTML element; the id specifies a unique format for that element.

  • Class:

    The class selector is written in CSS (as .id name) and then assigned to a group of HTML elements. The class attribute applies the same style to all of the relevant elements.

HTML

All documents must start with the tag: !DOCTYPE html.

The document itself begins with an "html" tag and ends with "/html" tag.

Each HTML document needs a "head" section demarcated by head tags. This section includes the website's title, link to a CSS stylesheet, and other important details that apply to the whole site

The visible part of the document is between "body" and "/body" tags.

Headings are defined with the h1 to h6 tags.

Paragraphs are defined with the p tag.

Links are defined with the "a" tag. The address is specified in the "href" attribute. Attributes are used to provide additional information about HTML elements.

HTML images are defined with the img tag. The source file (src), alternative text (alt), and size (width and height) are provided as attributes.

The div tag defines a division or section in an HTMl document

CSS

When tags like font, and color attributes were added to the HTML, it started a nightmare for developers. Development of large web sites, where fonts and color information were added to every single page, was long and expensive.

With CSS, you can change the look of an entire Web site by changing just one file!

The element selector selects elements based on the element name. For example, all style choices within a "p" selector will apply to all paragraph elements in the HTML document.

The id selector uses the id attribute of an HTML element to select a specific element.

The class selector selects elements with a specific class attribute.

You can also group selectors to minimize the code. To do this, separate each selector with a comma. The CSS styling will apply apply to elements whose position in the document matches the order of selectors.
